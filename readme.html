Instalación: la famosa instalación en 5 minutos

  1. Descomprima el paquete en un directorio vacío y cárguelo en su alojamiento web o algú otro destino en particular.
  2. Acceda al fichero wp-admin/install.php desde su navegador. Le llevará a través de un proceso para configurar el fichero wp-config.php con la información necesaria para conectarse a su base de datos.
    1. Si por alguna razón esto no funciona, no se preocupe. A veces no se logra que funcione en todos los alojamientos web. Acceda al fichero wp-config-sample.php con ayuda de algún editor de texto como WordPad o similar y agregue la información para conectarse a su base de datos.
    2. Guarde este fichero como wp-config.php y cárguelo.
    3. Acceda al fichero wp-admin/install.php desde su navegador.
  3. Una vez que el fichero esté configurado, el instalador creará las tablas necesarias para su sitio web en la base de datos. Si ocurre algún error, revise su fichero wp-config.php, e intente nuevamente. Si falla otra vez, por favor visite los foros de soporte, WordPress Perú, o WordPress Perú en Facebook, con toda la información que usted pueda brindar.
  4. Si no ha ingresado alguna contraseña, anote la contraseña que se le ha asignado. Si usted no ha proporcionado un nombre de usuario, éste será admin.
  5. El instalador le lleveará a la página de inicio de sesión. Ingrese con el nombre de usuario y contraseña que eligió durante la instalación. Si alguna contraseña fue generada para usted, entonces puede hacer clic en “Perfil” para modificarla.

Actualización

Con el actualizador automático

Si está actualizando desde la versión 2.7 o superior, puede utilizar este actualizador:

  1. Acceda al fichero wp-admin/update-core.php desde su navegador y siga las instrucciones.
  2. ¿Esperaba algo más? ¡Esto es todo!

Actualizando manualmente

  1. Antes de actualizar, asegúrese de realizar copias de seguridad de cualquier archivo que haya modificado, por ejemplo, del fichero index.php.
  2. Elimine los ficheros de su anterior versión de WordPress, guardando previamente aquellos que haya modificado.
  3. Cargue los nuevos ficheros.
  4. Dirija su navegador hacia el fichero /wp-admin/upgrade.php.

Migrando desde otros sistemas

WordPress puede importar contenido desde varios sistemas. Antes de usar nuestra herramienta de importación, usted necesita en primer lugar, tener WordPress instalado y funcionando como se describió anteriormente.

Requisitos del sistema

  • PHP versión 5.2.4 o superior.
  • MySQL versión 5.0 o superior.

Recomendaciones

  • PHP versión 5.4 o superior.
  • MySQL versión 5.5 o superior.
  • El módulo mod_rewrite de Apache.
  • Un enlace a wordpress.org en su sitio web.

Recursos en línea

Si alguna de sus preguntas no tiene respuesta en este documento, le sugerimos que aproveche los numerosos recursos de WordPress en la red:

El Codex de WordPress
El Codex es la enciclopedia de todo lo relacionado con WordPress. Es la fuente de información más detallada disponible sobre WordPress.
El Blog de WordPress
Aquí encontrará las últimas actualizaciones y noticias relacionadas con WordPress. Las recientes noticias aparecerán por defecto en su panel de administración.
Planeta WordPress
Planeta WordPress es un agregador de noticias que recopila entradas de blogs de WordPress por toda la web.
Foros de Soporte de WordPress
Si ha buscado por todas partes pero continúa sin hallar respuestas, en los activos foros de soporte cuenta con una amplia comunidad deseosa de ayudar. Para ayudarle, asegúrese de usar un título descriptivo e incluir en su pregunta tantos detalles como le sea posible.
WordPress Perú
También puedes solicitar ayuda en la página de WordPress Perú.
WordPress Perú en Facebook
Otra alternativa es la página oficial en la red social Facebook.
Canal IRC de WordPress
Existe un canal de chat utilizado por los usuarios de Wordpress para intercambiar opiniones y eventualmente solicitar ayuda. La página wiki que ve arriba puede orientarle. (irc.freenode.net #wordpress)

Notas finales

  • Si tiene alguna sugerencia, idea, comentario, o si encuentra (¡oh!) algún fallo, únase a nosotros en los Foros de Soporte.
  • WordPress tiene una robusta API que facilita la ampliación del código básico. Si usted es un desarrollador interesado en utilizarla, revise la documentación acerca de los plugins en el Codex. Generalmente, usted no necesitará modificar nada del código.
